2.  Describe the purpose of a DIV tag.

 A div is a container for HTML elements. It allows you to group similar content on a web page together and style it accordingly.

10.  Why do we use media queries?

 Media queries allow you to change the style of an element for different sizes of screens (mobile, desktop, etc) 

11. What is the main difference between a serif and a sans-serif font?

 A serif font has decorative lines on the edges of the letters in order to mimic handwriting. 
 A sans-serif font does not have these decorative lines, and is meant for screens.
